atom

open-source backend framework to accelerate your development.

about atom

atom is an open-source backend framework that reduces repetitive work through reusable modules, enabling faster backend development and iteration. it is designed for entrepreneurs to build and scale their ideas quickly with minimal backend work.

  • built with python, fastapi, postgresql, and redis for scalability.
  • optimized queries using raw sql for better control.
  • supports high-throughput pipelines with rabbitmq, lavinmq, or kafka.
  • one-click deployment and database setup.
  • easy integration with s3, ses, sns, and more.

coding paradigm

atom's codebase combines procedural and functional programming to offer the following benefits:

  • procedures for better global/local state access.
  • functions for deterministic behavior of the code.
  • supports modularity, scalability, and maintainability following solid principles.
  • ensures robust, high-performance development for large-scale applications.
  • efficient memory usage and data encapsulation.

important: atom's entire codebase is in a single file, main.py.

how to run atom

follow these steps to run atom locally:

  1. download the atom repo
  2. install the requirements using pip install -r requirements.txt
  3. create a .env file
  4. run python main.py

or run it using docker:

  1. build the docker image with docker build -t atom .
  2. run the container with docker run -p 8000:8000 atom
