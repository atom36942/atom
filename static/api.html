Notes

            
1. All endpoints are loaded live from /openapi.json.

            
2. Display values can be overridden from the internal override map.

            
3. Execution follows the live schema plus any manual override you define.

            
4. Unknown override keys are added to Query for GET and Body for non-GET methods.

        

        

            
Query Operators (GET /object-read)

            

                
• Comparison: =,val, ==,val, >,5, >=,2024-01-01

                
• List: in,val1|val2, not in,val1|val2, between,val1|val2

                
• Pattern: like,%val%, ilike,%Python%, ~,^admin (regex)

                
• Nulls: is,null, is not,null, is distinct from,0

                
• Arrays: contains,val1|val2, overlap,val1|val2, any,val

                
• JSONB: contains,key|val|str (or int/bool), exists,key

                
• Spatial: point,lon|lat|min_dist|max_dist
