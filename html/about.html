About

atom36942/atom is a modular backend framework combining functional and procedural styles, designed to speed up development with minimal boilerplate.

Key Features

  • Modular architecture
  • Pure functions
  • Supports Postgres, Redis, S3, Kafka
  • Production-ready APIs & jobs
  • Minimal boilerplate
  • Flexible business schema

Installation

git clone https://github.com/atom36942/atom.git
cd atom
pip install -r requirements.txt

Usage

python main.py
